# Hungarian Notation Used  
I use hungarian notation in this work since im developing w/o a fancy ide which just automagically does everything, this file includes the notation used throughout the files, incl. custom.

## Standard Notations  
| Type | Prefix |
-----------------
| string | sz |
| uint | u |
| int | i |

## Custom Notations
| Type | Prefix |
-----------------
| IniTypes | it |
